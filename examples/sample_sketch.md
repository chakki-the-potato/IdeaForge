# 냉장고 재료 기반 요리 추천 서비스

## 문제 의식
- 1인 가구인데 장을 한 번 봐도 뭘 먹을지 몰라서 재료를 썩혀버림
- 배민/쿠팡이츠만 돌려도 월 50만원 나감
- 레시피 블로그는 재료 리스트부터 요구해서 현실과 안 맞음

## 초안 스케치
- 냉장고 내부를 스마트폰으로 촬영
- AI가 재료를 인식
- 해당 재료로 만들 수 있는 요리 3~5개 추천
- 부족한 재료는 쿠팡 장바구니 연동
- 조리 난이도/소요시간 필터

## 타겟
- 20~30대 1인/2인 가구
- 요리 초보~중급

## 수익 모델
- 기본 무료
- 월 4,900원 프리미엄: 무제한 추천 + 식단 계획
- 쿠팡 제휴 커미션
